# ArquivoLAN

## Sobre
ArquivoLAN é um projeto de plataforma para inventário de arquivos em redes corporativas. A plataforma pretende permitir ao administrador descobrir:
  - Quais arquivos existem em determinada máquina
  - Quais diretórios crescem mais
  - Quais máquinas têm certos tipos de arquivos
  - etc.

## Tecnologias
- .NET 10
- C#
- ASP .NET Core
- Microsoft SQL Server
